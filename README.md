# Yoga-Fi — Yoga & Ayurveda (installable web app)

## Upload ALL of these files to your repo root (same folder)
- index.html            (the app)
- yoga.html             (identical copy)
- yoga.webmanifest      (PWA manifest)
- yoga-sw.js            (service worker — enables install/offline)
- yoga-icon-192.png     (app icon)
- yoga-icon-512.png     (app icon)
- yoga-apple-touch.png  (iOS home-screen icon)

## Add to Home Screen (uses the Aditi sunrise logo as the app icon)
- iPhone/iPad (Safari): Share -> Add to Home Screen.
- Android (Chrome): menu -> Install app / Add to Home screen.
Requires HTTPS — GitHub Pages provides this automatically.

## GitHub Pages
Settings -> Pages -> Deploy from a branch -> main / (root) -> Save.
